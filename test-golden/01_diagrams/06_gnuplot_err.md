# Gnuplot

We try out the Gnuplot integration when things go wrong...

## An invalid script given to gnuplot

``` {.gnuplot table="foo"}
set xtic 1
plott $foo using 1:2 with lines title "foo"
```

"plott" is invalid, it should be "plot".

::::: {data-table="foo" .hidden}
   X    Y
---- ----
   1   10
   2   12
   3    9
   4    3
   5   10
:::::
